# Helm Skills

Helm chart management for the Kagenti platform.

## Available Skills

| Skill | Description |
|-------|-------------|
| `helm:debug` | Debug chart issues - templates, values, hooks |

## When to Use

- **helm:debug**: Helm install/upgrade failures, template issues, value debugging

## Related Skills

- `kagenti:deploy` - Platform deployment using Helm
- `k8s:health` - Platform health after Helm changes
